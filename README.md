# Public Transport Token System

A blockchain-based public transportation payment system built on Stacks blockchain using Clarity smart contracts.

## Features
- Token-based payment system
- Usage-based governance tokens
- Route management
- Operator registration
- Governance proposals and voting

## Smart Contract Functions
- `mint-transport-tokens`: Mint new transport tokens (owner only)
- `purchase-tokens`: Buy transport tokens with STX
- `pay-for-ride`: Pay for transportation using tokens
- `add-route`: Add new transport routes (owner only)
- `register-operator`: Register as transport operator
- `create-proposal`: Create governance proposals
- `vote-on-proposal`: Vote on governance proposals

## Installation
1. Clone the repository
2. Install dependencies: `npm install`
3. Deploy contract using Stacks CLI

## Usage
Deploy the contract to Stacks testnet/mainnet and interact through web3 applications or CLI.
